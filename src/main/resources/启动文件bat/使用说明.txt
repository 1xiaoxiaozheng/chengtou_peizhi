========================================
   FACT 贷款管理系统启动脚本使用说明
========================================

【文件说明】
1. 启动FACT.bat
   - 前台运行，可以看到应用启动日志
   - 适合调试和查看运行状态
   - 关闭窗口会停止应用

2. 启动FACT-后台运行.bat
   - 后台运行，窗口最小化
   - 适合生产环境部署
   - 关闭窗口不会停止应用（需使用停止脚本）

3. 停止FACT.bat
   - 自动查找并停止FACT应用进程
   - 如果停止失败，可能需要管理员权限

【使用步骤】
1. 确保Java环境已配置（Java 1.8）
2. 确保JAR包路径正确：D:\springbooot\FACT-0.0.1-SNAPSHOT.jar
3. 双击运行对应的bat文件

【配置说明】
- 应用端口：8089
- 全量日志路径：D:\springbooot\Log\allLog
- 错误日志路径：D:\springbooot\Log\error
- 日志滚动：每日自动滚动，保留30天
- JVM内存：最小512MB，最大1024MB

【注意事项】
1. 如果JAR包路径不同，请编辑bat文件中的 JAR_PATH 变量
2. 如果Java环境变量未配置，请取消注释bat文件中的JAVA_HOME设置
3. 日志目录会在启动时自动创建，无需手动创建
4. 如需修改JVM参数，请编辑bat文件中的 JAVA_OPTS 变量
5. 日志文件每日自动滚动，格式：application.yyyy-MM-dd.log 和 error.yyyy-MM-dd.log

【常见问题】
1. 提示"JAR包不存在"
   - 检查JAR包路径是否正确
   - 确认JAR包文件是否存在

2. 提示"未找到Java环境"
   - 检查Java是否已安装
   - 检查环境变量PATH是否包含Java路径
   - 在命令行执行 java -version 验证

3. 应用启动失败
   - 查看全量日志：D:\springbooot\Log\allLog\application.log
   - 查看错误日志：D:\springbooot\Log\error\error.log
   - 检查端口8089是否被占用
   - 检查数据库连接配置是否正确

4. 无法停止应用
   - 使用任务管理器手动结束Java进程
   - 或使用管理员权限运行停止脚本
